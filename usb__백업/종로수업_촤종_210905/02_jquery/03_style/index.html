[jQuery 스타일 변경(선택자대신 변수로)]

  1. $('대상').css(); css 명령을 사용할때
  2. $('대상').show(); 숨어있는 요소를 보이게 해주는 명령
  3. $('대상').hide(); 보이는 요소를 숨기게 해주는 함수
  4. $('대상').toggle(); 한번클릭하면 요소가 보이고 또 한번 클릭하면 요소를 안보이게 해주는 함수
  5. $('대상').width(); 요소의 가로사이즈 조절할때
  6. $('대상').height(); 요소의 세로사이즈 조절할때
